## To Create a Personal Blog Website
- How to do: **Astro-based Theme** + GitHub + Cloudflare.
- Change the Astro-based Theme according to personal information and habits.
- **Git**: push the Blog Folder to a remote repository on GitHub.
- **Cloudflare**: distribute the repository to everywhere around the world.

Notes: All is free.
